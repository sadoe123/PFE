      🔍 Import Vues SQL — relations view_join
      
    

    


      
      

        
        
      


      
      

        

          Fetch automatique — OnePilot se connecte à ta base SQL Server et récupère automatiquement toutes les définitions de vues via sys.views, puis extrait les jointures.


          ✅ Aucune manipulation manuelle

          ✅ Toutes les vues analysées en une fois

          ✅ Confidence = 1.0 (jointures confirmées par le code)
        

        

      


      
      

        
Exécute cette requête dans SSMS et colle le résultat :

        
SELECT
    v.name          AS view_name,
    m.definition    AS sql_definition
FROM sys.views v
JOIN sys.sql_modules m ON v.object_id = m.object_id
WHERE v.type = 'V'
ORDER BY v.name
